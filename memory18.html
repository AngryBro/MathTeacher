Памятка по задачам на составление модели

Что такое задача?

  1. Задача – это конкретный набор числовых данных, связанных различными условиями.
  2. Составить пример означает привести один частный случай, в котором выполняются все условия задачи.
  3. Составить математическую модель означает найти все связи между числовыми данными
    и записать их в наиболее удобном для исследования виде.

Составление математической модели

  1. Модель задачи составляется из утверждений: уравнений, неравенств и т.д.
  2. Для использования неизвестных величин вводятся переменные.
  3. Смысл каждой переменной должен быть отражён в описании модели.
  4. Составление неочевидных из условия выражений должно быть объяснено в описании модели.
  5. За одну переменную можно обозначить только одно число.
  6. Числа, которые могут быть различными должны быть обозначены разными переменными.

Исследование математической модели

  1. Цель исследования модели – получение некоторого выражения.
  2. Общий принцип:
    – связывать выражения из модели
    – максимально уменьшить количество переменных в выражениях
    – максимально уменьшить количество возможных значений переменных
    – разбить один общий случай на все частные и достичь цели в каждом из них.
  3. В задачах про целые числа исследование проводится преимущественно
    с помощью свойств делимости и составления неравенств.
  4. Если количество всех возможных случаев ограничено, то их можно все рассмотреть.
  5. Если в процессе исследования модели во всех случаях получилось ложное утверждение,
    то ситуация в задаче невозможна.

Идея ПРИМЕР+ОЦЕНКА

  1. Применяется для поиска наибольшего или наименьшего значения величины.
  2. На шаге ПРИМЕР приводится конкретная ситуация, где данная величина принимает
    наибольшее или наименьшее значение.
  3. На шаге ОЦЕНКА показывается, что данная величина
    не может превышать или быть меньше найденного значения.
