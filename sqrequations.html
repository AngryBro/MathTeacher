Общего вида

Уравнение:

\(ax^2+bx+c = 0,~~a \neq 0\).

Решение:
\(D = b^2 - 4ac,~~~~\) \( x = \displaystyle \frac{-b \pm \sqrt{D}}{2a}. \)

Пример:

\(-2x^2 + 3x+4 = 0 ,\)

\(D = 3^2+4\cdot 2\cdot 4 = 32,\)

\( x = \displaystyle\frac{-3 \pm \sqrt{32}}{-4}\).

Второй чётный коэффициент

Уравнение:

\(ax^2+bx+c = 0,~~a \neq 0,~~b ~\vdots~ 2.\)

Решение:
\(\displaystyle\frac{D}{4} = \left(\frac{b}{2}\right)^2 - ac,~~~~\) \( x = \displaystyle \frac{-\frac{b}{2} \pm \sqrt{\frac{D}{4}}}{a}. \)

Пример:

\(3x^2 -4x-5 = 0 ,\)

\(\frac{D}{4} = 2^2+3\cdot 5 = 19,\)

\( x = \displaystyle\frac{2 \pm \sqrt{19}}{3}\).

Первый коэффициент 1

Уравнение:

\(x^2+bx+c = 0.\)

Решение:

\(\begin{cases} x_1 \cdot x_2 = c \\ x_1+x_2=-b \end{cases}~,~~~~\) \( \left[ \begin{array}{l} x = x_1 \\ x = x_2 \end{array} \right.~ \).

Пример:

\(x^2 -5x+6 = 0 ,\)

\( \begin{cases} x_1 \cdot x_2 = 6 \\ x_1 + x_2 = 5 \end{cases} \Leftarrow \begin{cases} x_1 = 2 \\ x_2 = 3 \end{cases}~, \)

\( \left[ \begin{array}{l} x = 2 \\ x = 3 \end{array} \right.~ \).

Последний коэффициент \(\pm 1\)

Уравнение:

\(ax^2+bx \pm 1 = 0,~~a \neq 0.\)

Решение:

\({\tilde x}^2+b\tilde x \pm a = 0\),

\( \left[ \begin{array}{l} x = \frac{\tilde x_1}{a} \\ x = \frac{\tilde x_2}{a} \end{array} \right.~ \).

Пример:

\(6x^2 +x-1 = 0 ,\)

\( {\tilde x}^2 + \tilde x - 6 = 0,\)

\( \left[ \begin{array}{l} \tilde x = 2 \\ \tilde x = -3 \end{array} \right.~, \)

\( \left[ \begin{array}{l} x = \frac{2}{6} \\ x = \frac{-3}{6} \end{array} \right.~ \).

Сумма коэффициентов 0

Уравнение:

\(ax^2+bx + c = 0,~~a \neq 0,~~a+b+c=0.\)

Решение:

\( \left[ \begin{array}{l} x = 1 \\ x = \frac{c}{a} \end{array} \right.~ \).

Пример:

\(3x^2 -7x+4 = 0 ,\)

\( \left[ \begin{array}{l} x = 1 \\ x = \frac{4}{3} \end{array} \right.~ \).

Знакочередующаяся сумма коэффициентов 0

Уравнение:

\(ax^2+bx + c = 0,~~a \neq 0,~~a-b+c=0.\)

Решение:

\( \left[ \begin{array}{l} x = -1 \\ x = -\frac{c}{a} \end{array} \right.~ \).

Пример:

\(3x^2 +7x+4 = 0 ,\)

\( \left[ \begin{array}{l} x = -1 \\ x = -\frac{4}{3} \end{array} \right.~ \).
