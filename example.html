Задача 1

На доске написано 100 различных натуральных чисел с суммой 5100.
а) Может ли быть записано число 250?
б) Можно ли обойтись без числа 11?
в) Какое наименьшее количество чисел кратных 11 может быть записано?

Решение

a) Предположим может.
Записаны различные натуральные числа: \( a_1;~ a_2;~\ldots ;~a_{99};~250. \)
\(a_1 + a_2 + \ldots + a_{99}+250 = 5100.\)
\(a_1 + a_2 + \ldots + a_{99} = 4850 \).
\( 4850 = a_1 + a_2 + \ldots + a_{99} \geq 1+2+3+\ldots + 99 = 4950. \)
\( 4850 \geq 4950 ~-\) противоречие.
Ответ: нет.

б) Предположим можно.
Записаны натуральные различные числа без 11: \( a_1;~ a_2;~\ldots ;~a_{100} \).
\( a_1 + a_2 + \ldots + a_{100} = 5100 \).
\( 5100 = a_1 + a_2 + \ldots + a_{100} \geq 1+2+ \ldots + 10 + 12 +13 + \ldots + 101 = 5140\).
\(5100 \geq 5140 ~-\) противоречие.
Ответ: нет.

в)
Пример:
Числа от 1 до 102 и 111, за исключением 99, 88, 77.
Всего чисел 100, кратные 11: 11, 22,..., 66 – 6 штук.
Сумма \( 1+\ldots+102 + 111 -99-88-77 = 5100. \)

Оценка:
Предположим, что чисел кратных 11 меньше 6.
\(x~ -\) количество чисел делящихся на 11.
\( a_1 < a_2 < a_3 < \ldots < a_{91}< \ldots < a_{100-x} ~- \) числа не делящиеся на 11.
\( b_1 < b_2 < \ldots < b_x ~- \) числа делящиеся на 11.
\( a_1 + a_2 + \ldots + a_{100-x} + b_1 + b_2 + \ldots + b_x = 5100. \)
\( b_1;~b_2;\ldots ;b_x ~\vdots~ 11 \).
\( a_1;~a_2;~\ldots ;a_{100-x} ~\not\vdots~ 11\).
\( x<6 \). (*)
\( a_1 + a_2 + \ldots + a_{91} \geq 1+2+ \ldots +100 -(99+88+ \ldots + 11) = 4555\).
\( a_{92} + a_{93} + \ldots + a_{100-x} \geq \underbrace{101+102+ \ldots }_{9-x~штук} = \frac{(202+(8-x))(9-x)}{2} = \frac{1}{2} (210-x)(9-x) \).
\( b_1 + b_2 + \ldots + b_x \geq \underbrace{11+22+\ldots}_{x~штук} = \frac{11}{2} x(x+1) \).
\( a_1 + \ldots + a_{91} + \ldots + a_{100-x} + b_1 + \ldots + b_x \geq 4555 + \frac{1}{2} (210-x)(9-x) +\frac{11}{2} x(x+1) \).
\( 5100 \geq 4555+ \frac{1}{2}( (210-x)(9-x) +11x(x+1) ) \).
\( 1090 \geq 1890+x^2 - 219x + 11x^2+11x \Leftrightarrow 12x^2 - 208x + 800 \leq 0 \).
\( 3x^2 - 52x + 200 \leq 0 \Rightarrow x \geq \frac{26 - \sqrt{76}}{3} > \frac{26-\sqrt{121}}{3} = 5 \).
\( x>5 \Rightarrow x \geq 6 ~- \) противоречие с (*).
Ответ: 6.
