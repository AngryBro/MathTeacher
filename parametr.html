Графический

Постановка задачи

Определить сколько решений \((x,y)\) имеет выражение \(F(x,y,a)\)
в зависимости от параметра \(a\).

Решение

  1. Преобразование \(F(x,y,a) \Leftrightarrow \begin{cases} G(x,y) \\ A(x,y,a) \end{cases} \).
  2. Даём описание графику \(G\), строим его.
  3. Даём описание графику \(A\), определяем на что влияет \(a\).
  4. Перебираем все \(a\) от \(\operatorname{min}(a)\) до \(\operatorname{max}(a)\).
    Строим график \(A\) при \(a=\operatorname{min}(a)\), медленно увеличиваем \(a\),
    следим за тем, как меняется график \(A\).
    Всякий раз, когда количество общих точек \(A\) и \(G\) меняется,
    рисуем график \(A\) на том моменте, начиная с которого / в котором / заканчивая которым
    количество общих точек с графиком \(G\) меняется.
    Подписываем сколько общих точек в каждом граничном случае.
  5. В каждом зарисованном граничном случае находим \(a\).
  6. На прямой отмечаем все найденные \(a\), подписываем сколько решений при таких \(a\),
    Если решений нужное количество, точка закрашенная, иначе выколотая.
  7. По графику считаем количество решений на промежутках,
    подписываем, закрашиваем нужные.
  8. В ответ выписываем закрашенные промежутки.

Аналитический

Постановка задачи

Определить сколько решений \(x\) имеет выражение \(F(x,a)\)
в зависимости от параметра \(a\).

В качестве \(x\) может выступать вектор.

Решение

  1. Преобразование \(F(x,a) \Leftrightarrow \left[ \begin{array}{l} G_1(x,a) \\ G_2(x,a) \\ \ldots \\ G_n(x,a) \end{array} \right. \)
  2. Определяем возможные решения \(x_i\).
  3. Выполняется поиск множеств:
    \(A_i = \{a ~:~ x_i~является~решением\}\)
    \(A_{\infty} = \{a ~:~ решений~бесконечно\}\)
    \(B_{i,j} = \{a~:~x_i=x_j,~i < j \} \).
  4. Возможны следующие случаи:

    \(\large Простой\)
    \( G_i(x,a) \Leftrightarrow \begin{cases} x = x_i(a) \\ H_i(x,a) \end{cases} \)

    Тогда \(H_i(x_i,a) \Leftrightarrow a \in A_i\).

    \(\large Сложный\)
    Определяем \(a\), при которых \(G_i(x,a) \Leftrightarrow \begin{cases} x=x_i(a) \\ H_i(x,a) \end{cases} \).
    Исследуется как простой случай.

    При остальных \(a\) проверяем что получится. Либо решений не будет, либо будет бесконечно.
  5. Находим множества \(B_{i,j}= \{a~:~x_i=x_j,~i < j \}\).
    Для этого решаем системы \( \begin{cases} G_i(x,a) \\ G_j(x,a) \end{cases} \)
  6. Рисуем \(n\) осей \(a\). На оси для \(x_i\) рисуем множество \(A_i\).
  7. На осях для \(x_i\) и \(x_j\) соединяем дугой числа из множества \(B_{i,j}\).
  8. Рисуем ось \(a\). Отмечаем все граничные числа из всех множеств \(A_{\infty},~A_i,~B_{i,j}\).
    Подписываем сколько решений в каждой точке.
    Там где нужное число решений, точка закрашенная, иначе выколотая.
  9. Подписываем сколько решений на промежутках, нужные закрашиваем.
  10. В ответ выписываем закрашенные промежутки.

Замена

  1. Добавление/исключение переменной.

    \( \begin{cases} F(x,y,a) \\ y = f(x,a) \end{cases} \Longrightarrow F\left(x,f(x,a),a)\right) \)

    Количество решений при таком переходе сохраняется.
  2. Переход к новой переменной

    \(F(f(x),a) \overset{t=f(x)}{\Longleftrightarrow} F(t,a)\)

    В координатах \(Oxt\) рисуется график \(t=f(x)\).
    По нему строится таблица для множеств \(t\) и количества решений \(x\).

    Множество \(t\)Количество \(x\)
    \(T_1\)\(k_1\)
    \(T_2\)\(k_2\)
    \(\ldots\)\(\ldots\)
    \(T_n\)\(k_n\)
    \((T_1 \cup T_2 \cup \ldots \cup T_n = \mathbb{R})\)

    При решении задачи для \(F(t,a)\) количество решений для \(x\)
    считается с помощью таблицы. Обратная замена не делается.
  3. Переход к новому параметру

    \(F(x,f(a)) \overset{b = f(a)}{\Longleftrightarrow} F(x,b)\)

    Найти \(b\) для задачи с \(F(x,b)\), значения \(a\) получить обратной заменой.

Квадратный трёхчлен

Постановка задачи

Исследовать множества решений выражения \(ax^2+bx+c \vee 0\)
на заданных множествах, ограниченных числами \(x_i\).

Решение

  1. Обозначаем и считаем:
    • \( f(x) = ax^2+bx+c \)
    • \( D = b^2-4ac \)
    • \( x_в = -\displaystyle\frac{b}{2a} \)
    • \( f(x_i) = ax_i^2+bx_i+c \)
  2. Перебор всех случаев:
    • \(a = 0,~a < 0,~a>0\)
    • \(D < 0,~D = 0,~D>0\)
    • \(x_в \vee x_i\)
    • \(f(x_i) \vee 0\)
    Реализация перебора:

    \(\tiny n_1)\normalsize ~a \vee 0\)
    \(\tiny n_1.n_2)\normalsize~ D\vee 0\)
    \(\tiny n_1.n_2.n_3)\normalsize ~x_в \vee x_i\)
    \(\tiny n_1.n_2.n_3.n_4)\normalsize ~f(x_i)\vee 0\)
    \(\tiny n_1.n_2.n_3.n_4.n_5)\normalsize ~f(x_j)\vee 0\)
    Если случай не подходит, то вложенные случаи не рассматриваются.
    Если случай может подойти, то рассматриваются вложенные случаи.

    Замечания по перебору:
    • Случай \(a=0\) не содержит подслучаев.
    • Случай \(D < 0 \) не содержит подслучаев.
    • Случай \(D=0\) содержит только подслучаи \(x_в\vee x_i\).
    • Случай, выражающийся ложным при любых значениях
      параметров выражением не рассматривается.
  3. Выбрав подходящие случаи, составляется выражение \(F(a,D,x_в,f(x_i))\),
    решение которого является ответом.

Симметрия

Постановка задачи

Найти значения параметра \(a\), при каждом из которых
\(F(x,a)\) имеет \(n\) решений.

В качестве \(x\) может выступать вектор.

Решение

  1. Определяем тип симметрии.
    Если \(x\) является решением, то \(f_1(x),~f_2(x),\ldots f_k(x)\) также являются решениями.
    (далее считаем \(f_0(x)=x\)).
  2. Проверяем, так ли это:
    \(F(f_i(x),a) \Leftrightarrow F(x,a)\).
  3. Если \(n < k+1\), рассматриваем совпадения решений:
    \( \begin{cases} F(x,a) \\ f_i(x) = f_j(x) \end{cases} \).
  4. При каждом найденном \(a\) решаем \(F(x,a)\).
  5. В ответ выписываем \(a\), при которых ровно \(n\) решений.

Оценка

Постановка задачи

Исследовать количество решений \(F(x,a)\)
в зависимости от параметра \(a\).

В качестве \(x\) может выступать вектор.

Решение

  1. Используя свойства выражений, составляем такую систему \(G(a) \Leftarrow F(x,a)\),
    решением которой является конечное множество значений \(a\).
  2. При каждом найденном \(a\) решаем \(F(x,a)\).
  3. Часто используются следующие свойства:
    • \( t^{2n},~|t| \geq 0 \)
    • \( -1 \leq \sin{x},~\cos{x} \leq 1 \)
